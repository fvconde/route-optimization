Como funciona a otimização?

O algoritmo genético avalia e penaliza as rotas com base nas seguintes restrições operacionais:

  • Distância Total: Busca a rota VRP mais curta agregada para os veículos.
  • Capacidade Máxima: Penaliza veículos sobrecarregados (peso/volume).
  • Autonomia: Penaliza fortemente se a distância exceder o limite do veículo.
  • Janelas de Tempo (SLA): Punição se a entrega atrasar com base em sua prioridade.
  • Horário Comercial: Punição drástica se a rota avançar além das 18:00h.
  • Prioridade de Entregas: Penaliza se entregas urgentes forem movidas para o fim da fila.
